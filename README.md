# Papr
A wallpaper manager that specializes in categorizing and storing wallpapers.

## How to build
Run these commands before using CMake to install [yaml-cpp](https://github.com/jbeder/yaml-cpp):
```
git submodule init
git submodule update
```

CMake: (Requires C++17)
```
mkdir build
cd build
cmake ..
cmake --build .
```
